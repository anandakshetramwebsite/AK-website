# Ananda Kshethram

Premium eco-retreat website for Ananda Kshethram — Hyderabad's 100% pure vegetarian farm sanctuary.

## Stack

- **Next.js 15** (App Router)
- **React 19**
- **Framer Motion** — cinematic animations
- **Tailwind CSS v4** — design system tokens
- **Vercel** — deployment target

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy to Vercel

```bash
npx vercel
```
